Primeras lecciones de JavaScript

Imprimir en consola

Para imprimir un valor en la consola utilizamos:
console.log("Hola, este es un mensaje creado para aprender JS");

y aparece lo que hayamos puesto entre el paréntesis.
(Hola, estre es un mensaje creado para aprender JS)

Que es una variable:

Es un espacio de almacenamiento

Declaramos la variable con valor
Ej. var x = 5;

Imprimir en consola el valor de la variable
console.log(x);
(5)

Para declarar una variable sin darle valor
var a;
Console.log(a);
(undefined)

Se recomienda usar nombres de las variables decriptibas Usando la convención "camelcase":

  1. primera letra minuscula
  2. primera letra de las palablas consecuentes en mayúsculas

Ej. miVariable, estaEsMiVariable, variableDescriptiva

Operador de Asignación

El signo de '=' asigna el valor a nuestra variable, tanto cuando no está definida (undefined) como cuanto ya esta definida y queremos modificarla.

Podemos darlo un valor o modificar la variable ya asignada dandole otro valor:
x = 15

Volvemos a Imprimir y ahora el valor se actualizó
console.log(x);
(15)

Asignar el valor de una variable a otra variable

Podemos asignar el valor de una variable usando otra ya previamente asignada: var a = 5;
var b = a;

console.log(a);
console.log(b);

También existe esta otra manera:
var a = 5;
var b;

b = a;

console.log(a);
console.log(b);
